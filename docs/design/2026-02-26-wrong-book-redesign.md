# 错题本功能重新设计

> **创建日期**: 2026-02-26
> **设计目标**: 优化错题本逻辑，提供更灵活的错题管理和复习方式

---

## 📋 需求变更

### 1. 去除功能
- ❌ 删除"答对三次自动移除"逻辑
- ❌ 删除相关开关和配置

### 2. 新增分类方式
- ✅ 按刷题任务分类（原有）
- ✅ 按题型分类（新增）

### 3. 功能重构

#### 按刷题任务分类
```
刷题任务
├── 错题回顾（详情查看模式）
├── 错题复习（刷题重做模式）
└── 易掌握题集（已掌握题目）
```

### 二、错题回顾页面（详情查看模式）

#### 2.1 页面结构
```
┌─────────────────────────────────────┐
│ ← 错题回顾                          │
├─────────────────────────────────────┤
│ 1 / 15                错误2次       │
├─────────────────────────────────────┤
│                                      │
│ [题型标签] 单选题                    │
│                                      │
│ 题干内容...                          │
│                                      │
│ ○ A. 选项A  [你的选择]              │
│ ● B. 选项B  [正确答案]              │
│ ○ C. 选项C                          │
│ ○ D. 选项D                          │
│                                      │
│ 💡 解析                              │
│ 解析内容...                          │
│                                      │
│ 📊 作答记录                          │
│ • 2026-02-20 10:30 - 答错（选A）   │
│ • 2026-02-18 15:20 - 答错（选A）   │
│                                      │
├─────────────────────────────────────┤
│ [上一题] [重做本题] [下一题]        │
│ [标记已掌握] [移除错题集]           │
└─────────────────────────────────────┘
```

#### 2.2 按钮功能

**重做本题**：
- 进入答题模式，重新作答该题
- 答题后显示结果反馈
- 答对后询问是否标记为已掌握

**标记为已掌握**：
- 将题目移入"易掌握题集"
- 从错题列表中移除
- Toast 提示"已标记为掌握"

**移除错题集**：
- 二次确认弹窗
- 从错题本中彻底删除
- Toast 提示"已移除"

### 三、错题复习页面（刷题重做模式）

#### 3.1 页面结构
```
┌─────────────────────────────────────┐
│ ← 错题复习 - 数据结构与算法         │
├─────────────────────────────────────┤
│ 1 / 15                              │
├─────────────────────────────────────┤
│                                      │
│ [题型标签] 单选题                    │
│                                      │
│ 题干内容...                          │
│                                      │
│ ○ A. 选项A                          │
│ ○ B. 选项B                          │
│ ○ C. 选项C                          │
│ ○ D. 选项D                          │
│                                      │
│ [提交答案]                           │
│                                      │
│ --- 答题后显示 ---                   │
│                                      │
│ ✓ 回答正确！                         │
│                                      │
│ 💡 解析                              │
│ 解析内容...                          │
│                                      │
│ [标记已掌握] [下一题]               │
└─────────────────────────────────────┘
```

#### 3.2 交互流程

1. **答题前**：
   - 显示题目和选项
   - 提交答案按钮

2. **答题后**：
   - 显示答题结果（正确/错误）
   - 显示正确答案和解析
   - 显示"标记已掌握"按钮
   - 显示"下一题"按钮

3. **标记已掌握**：
   - 点击后移入易掌握题集
   - 自动跳转到下一题

4. **完成所有题目**：
   - 显示复习结果统计
   - 正确率、用时等数据
   - 返回错题本按钮

- Toast 提示"已移回错题集"

## 🎨 UI 设计要点

### 1. 颜色规范
- **错题**: #F53F3F (红色)
- **已掌握**: #00B42A (绿色)
- **进度条**: #00B96B (主色)

### 2. 图标使用
- 错题回顾: 📖 (book)
- 错题复习: 🔄 (replay)
- 易掌握题集: ✅ (success)
- 移除: 🗑️ (delete)

### 3. 交互反馈
- 标记已掌握: Toast + 动画效果
- 移除错题: 二次确认弹窗
- 答题正确: 绿色成功提示
- 答题错误: 红色错误提示
