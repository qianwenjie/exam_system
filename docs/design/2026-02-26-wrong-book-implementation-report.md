## 🔄 数据迁移

### 旧数据结构
```javascript
wrongBook: {
  'q001': {
    wrongCount: 2,
    correctCount: 1,
    lastWrongTime: '...',
    mastered: false,
    sourceTaskId: 'task001',
    taskName: '数据结构',
    question: {...},
    userAnswer: 'A'  // 单个答案
  }
}
```

### 新数据结构
```javascript
wrongBook: {
  'q001': {
    qId: 'q001',
    wrongCount: 2,
    lastWrongTime: '...',
    mastered: false,
    masteredTime: null,
    sourceTaskId: 'task001',
    sourceTaskName: '数据结构',
    question: {...},
    userAnswers: [  // 答案数组
      {
        answer: 'A',
        isCorrect: false,
        answeredAt: '2026-02-20T10:30:00'
      }
    ]
  }
}
```

### 兼容性处理
- 旧数据会自动兼容（缺失字段会在下次答题时补全）
- 建议用户清除 LocalStorage 重新开始（可选）

## 📝 使用说明

### 1. 错题本首页
1. 打开错题本，默认显示"按任务分类"
2. 点击顶部标签可切换到"按题型分类"
3. 每个分类卡片显示错题数和已掌握数
4. 点击三个按钮进入不同功能

### 2. 错题回顾
1. 逐题查看错题详情
2. 查看历史作答记录
3. 点击"重做本题"进入单题重做
4. 点击"标记已掌握"移入易掌握题集
5. 点击"移除错题集"彻底删除

### 3. 错题复习
1. 以刷题模式重做所有错题
2. 答题后查看结果和解析
3. 答对后可标记已掌握
4. 完成所有题目后查看统计

### 4. 易掌握题集
1. 查看已标记为掌握的题目
2. 点击"查看详情"查看完整内容
3. 点击"取消掌握"移回错题集

### 5. 单题重做
1. 专注答题界面
2. 答对后可标记已掌握并返回
3. 答错后可重试或返回
